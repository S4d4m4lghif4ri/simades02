You are a senior software engineer specializing in Google Apps Script, web applications, and database system design.

Create a complete responsive web application called **SIMADES (Sistem Manajemen Administrasi Desa)**.

This system is used by a village government office to manage administration, document workflows, task management, village activities, and financial records.

The application must run as a **Google Apps Script Web App** with **Google Spreadsheet as the database** and **Google Drive as file storage**.

The UI must be modern, simple, and mobile friendly.

TECH STACK

Frontend:
HTML
CSS
TailwindCSS
Vanilla JavaScript

Backend:
Google Apps Script

Database:
Google Sheets

File Storage:
Google Drive

GENERAL SYSTEM REQUIREMENTS

1. The system must support role-based login.
2. The UI must use a sidebar dashboard layout.
3. The application must load fast and avoid excessive API calls.
4. Use asynchronous communication with google.script.run.
5. Use modular Apps Script functions for each module.
6. All files uploaded must be saved to Google Drive and linked in the spreadsheet.
7. Each record must have a unique ID.
