
  # Web App

  This is a code bundle for Web App. The original project is available at https://www.figma.com/design/uyg85jsZ8BfN4WTW4ZIfNb/Web-App.

  ## Running the code

  Run `npm i` to install the dependencies.

Copy `.env.example` to `.env` and fill in:

- `VITE_SUPABASE_URL`
- `VITE_SUPABASE_ANON_KEY`

  Run `npm run dev` to start the development server.
